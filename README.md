暗号通貨の価格を監視し、アラート通知や自動取引を行うシステムです。


---


##  主な機能

###  価格アラート
- 任意の価格条件を設定し、条件に達したときにLINEに通知する

###  自動取引
- 売買条件を設定し、条件達成時に自動で注文する
- 注文結果（成功/失敗）をLINEに通知する

### 各種設定
- LINE通知するためのMessagingAPIのキー情報を設定。
- 自動取引用のGMOコインAPIのキー情報を設定。
